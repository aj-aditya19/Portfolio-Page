# 🚀 Quick Start Guide - Your New Portfolio\n\n## Installation & Setup\n\n### 1. Install Dependencies\n`bash\nnpm install\n`\n\n### 2. Run Development Server\n`bash\nnpm run dev\n`\nOpen your browser and go to `http://localhost:5173`\n\n### 3. Build for Production\n`bash\nnpm run build\n`\n\n### 4. Preview Production Build\n`bash\nnpm run preview\n`\n\n---\n\n## 🎨 Key Features\n\n### 1. **Dark/Light Theme**\n- Click the **sun/moon icon** in the header to toggle theme\n- Your preference is automatically saved\n- The portfolio respects your system preference on first visit\n\n### 2. **Responsive Design**\n- Works perfectly on mobile, tablet, and desktop\n- Test it by resizing your browser window\n- All elements are touch-optimized for mobile\n\n### 3. **Smooth Navigation**\n- Header navigation links scroll smoothly to sections\n- Mobile hamburger menu available on screens < 768px\n- Quick navigation footer with smooth scroll\n\n### 4. **Modern Animations**\n- Hover effects on cards\n- Gradient animations on text\n- Smooth color transitions\n- Professional micro-interactions\n\n---\n\n## 📋 What's New?\n\n### New Components\n✅ **Header Component**\n- Sticky navigation bar\n- Theme toggle button\n- Mobile hamburger menu\n- Professional branding\n\n✅ **Footer Component**\n- Quick navigation links\n- Social media links\n- Year auto-update\n- Animated heart icon\n\n### New Context\n✅ **Theme Context**\n- Global theme management\n- Local storage persistence\n- System preference detection\n\n### New Styling System\n✅ **Global CSS with Theme Variables**\n- Light & dark mode support\n- Professional color palette\n- Semantic variable names\n- Responsive typography\n\n---\n\n## 🎯 Customization\n\n### Change Your Name/Title\n**File:** `src/assets/pages/About.jsx`\n`jsx\n<div className=\"name\">Your Name</div>\n<div className=\"title\">Your Title</div>\n`\n\n### Update Social Links\n**File:** `src/components/Footer.jsx`\n`jsx\nconst socialLinks = [\n  { name: \"GitHub\", url: \"https://github.com/yourprofile\", icon: \"github\" },\n  // Add your links here\n];\n`\n\n### Change Colors\n**File:** `src/styles/global.css`\n`css\n:root[data-theme=\"light\"] {\n  --accent-primary: #your-color;\n  --accent-secondary: #your-color2;\n  /* ... more variables */\n}\n`\n\n### Add New Sections\n1. Create a new component in `src/assets/pages/YourComponent.jsx`\n2. Create corresponding CSS in `src/assets/styles/YourComponent.css`\n3. Import in `src/assets/pages/Home.jsx`\n4. Add section ID and navigation link\n\n---\n\n## 📱 Responsive Breakpoints\n\nThe portfolio is optimized for:\n- **Mobile:** < 480px\n- **Tablet:** 480px - 768px\n- **Desktop:** 768px - 1200px\n- **Large Desktop:** > 1200px\n\n---\n\n## 🔗 Important Files\n\n| File | Purpose |\n|------|----------|\n| `src/App.jsx` | Main app with routing and theme provider |\n| `src/context/ThemeContext.jsx` | Theme management |\n| `src/components/Header.jsx` | Header & navigation |\n| `src/components/Footer.jsx` | Footer section |\n| `src/styles/global.css` | Global styles & theme variables |\n| `index.html` | HTML entry point |\n| `package.json` | Dependencies & scripts |\n\n---\n\n## 💡 Tips\n\n1. **Theme Testing**\n - Check browser console for any errors\n - Verify localStorage is working (DevTools → Application → LocalStorage)\n\n2. **Mobile Testing**\n - Use browser DevTools to test responsive design\n - Test on actual mobile devices\n - Check touch interactions\n\n3. **Performance**\n - Optimize images for web\n - Use lazy loading for large images\n - Check Lighthouse score (DevTools → Lighthouse)\n\n4. **Accessibility**\n - Test with keyboard navigation (Tab key)\n - Check color contrast (DevTools → Accessibility)\n - Screen reader testing\n\n---\n\n## 🐛 Troubleshooting\n\n### Theme not persisting?\n- Check if localStorage is enabled\n- Clear browser cache and reload\n- Check browser console for errors\n\n### Mobile menu not working?\n- Ensure JavaScript is enabled\n- Check browser console for errors\n- Try different browser\n\n### Styles not applying?\n- Clear CSS cache (Ctrl+Shift+R or Cmd+Shift+R)\n- Check if CSS files are imported correctly\n- Verify file paths are correct\n\n---\n\n## 📚 Resources\n\n- [React Documentation](https://react.dev)\n- [CSS Variables Guide](https://developer.mozilla.org/en-US/docs/Web/CSS/--*)\n- [Responsive Design Guide](https://web.dev/responsive-web-design-basics/)\n- [Vite Documentation](https://vitejs.dev)\n\n---\n\n## 🎉 You're All Set!\n\nYour portfolio now has:\n- ✅ Dark/Light theme support\n- ✅ Fully responsive design\n- ✅ Modern animations & effects\n- ✅ Professional styling\n- ✅ Smooth navigation\n- ✅ Optimized performance\n\n**Happy coding!
